## AUC Endpoints
- Long axial length: `AL >= 26 mm`.
- Any myopia: `SE <= -0.50 D`.
- High myopia: `SE <= -6.00 D`.

Model 5 was added for fair morphology comparison: `Clinical + OCTA_nonflow`, where OCTA_nonflow = `CT + CVI` and excludes flow-related `CCFA`.
